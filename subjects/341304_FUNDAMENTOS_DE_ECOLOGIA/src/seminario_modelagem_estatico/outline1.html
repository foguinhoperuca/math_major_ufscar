Conceitos

  • Definição – “Formulação pela qual se faz previsões”
  • Ilustração:
  • A equação da função horária é:
  • S= S0 + V x T para todo T ≥ 0 ; onde:
  • S = posição do móvel (m);
  • S0 = posição inicial do móvel (m);
  • V = velocidade do móvel (m/s);
  • T = tempo (s).

Conclusão

  • “Um modelo matemático nunca é uma representação completamente precisa de uma situação física – é uma idealização. Um bom modelo simplifica a realidade o bastante para permitir cálculos matemáticos, mantendo, porém uma precisão suficiente para conclusões apreciáveis. É importante entender as limitações do modelo. A palavra final está com a Mãe Natureza”.
  • James Stewart, in: ref. [1] – pág. 25.
